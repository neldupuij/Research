# Volatility Spillovers — EU Financial Institutions

## Lancer
make run

## Sorties
- CSV : `reports/exports/` (ex. `total_connectedness.csv`, `net_spillovers.csv`)
- Figures : `figures/` (ex. `total_connectedness.png`, `net_spillovers.png`)

## Environnements
- Restaurer : `make restore`
- Mettre à jour les locks : `make lock`
